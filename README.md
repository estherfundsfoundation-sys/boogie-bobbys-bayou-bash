# Boogie Bobby's Bayou Bash

Static public website for Bobby Vincent's annual birthday celebration and family reunion.

## Annual updates

Edit `index.html` to replace all “To Be Announced” event details, enable the RSVP button once a secure RSVP system is connected, and add approved family images in `assets/`.

## Important next build

The public site is ready to deploy. Before turning on RSVP, photo uploads, guestbook messages, recipes, or announcements, connect a private Supabase project. That will provide family sign-in, administrator approval, secure photo storage, CSV RSVP exports, and moderation so no unapproved information appears publicly.

The New Orleans guide uses official/destination links and should be checked again once the event date and venue are confirmed.
